# Boba Network Token Visualizer

Visualizer Token based on [Boba Network Token List](https://github.com/bobanetwork/token-list).

## Available Features

`Filter by Network`
`Serch by Token Name`
`Serch by Ticker`
`Serch by ChainId`
`Serch by Symbol`
`Serch by Decimals`


## Instalation / Usage

#### yarn
#### yarn start
